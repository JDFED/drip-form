## 配置
该字段配置在`schema`中：

```json title="schema" {12,13,14}
{
      "type": "string",
      "title": "输入框",
      "ui": {
        "type": "text",
        "placeholder": "请输入",
        "multiline": 2,
        "showFontNum": false,
        "disabled": false,
        "allowClear": true
      },
      "fieldKey": "text_gov2yB"
    }
```

:::info 提示

组件具体配置依赖于所使用的UI库，DripForm仅作示例，具体可参考UI库对应文档。

:::

## API
| 参数        | 说明     | 类型     |  是否必填 | 默认值 |
|:-----------|:--------------------------------------------------------------|:--------|:---|:---|
| filedKey      | 表单数据存放字段（必须唯一）| string   | 必填 | - |
| allowClear      | 可以点击清除图标删除内容 | boolean   | 非必填 | false |
| disabled      | 值为 `true` 时，输入框为禁用状态 | boolean   | 非必填 | false |
| multiline      | 文本行数 | boolean   | 非必填 | false |
| onChange      | 输入框内容变化时的回调 | function(e)  |  非必填 | - |
| placeholder      | 输入框占位文本 | string   | 非必填 | 请输入 |
| showCount      | 是否展示字数 | boolean | 非必填 | false |
