## 配置
该字段配置在`schema`中：

```json title="schema" {12,13,14}
{
    "validateTime": "submit",
    "type": "array",
    "title": "多选框",
    "default": "shanghai",
    "ui": {
        "type": "checkbox",
        "needCheckAll": true,
        "options": [
            {
            "label": "shanghai",
            "value": "shanghai"
            },
            {
            "label": "beijing",
            "value": "beijing"
            },
            {
            "label": "shenzhen",
            "value": "shenzhen",
            "disabled": true
            }
        ]
    }
}
```
:::info 提示

组件具体配置依赖于所使用的UI库，DripForm仅作示例，具体可参考UI库对应文档。

:::

## API
| 参数        | 说明     | 类型     |  是否必填 | 默认值 |
|:-----------|:--------------------------------------------------------------|:--------|:---|:---|
| fieldKey      | 表单数据存放字段（必须唯一） | string   |必填| - |
| options | 可选数据源              | options[ ] | 必填 | - |
| needCheckAll | 判断是否使用全选 | boolean | 非必填| false |
| onChange | 变化时回调函数 | function(e:Event) |  非必填  | - |
