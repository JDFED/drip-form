## 配置
该字段配置在`schema`中：

```json title="schema" {12,13,14}
{
    "type": "boolean",
    "title": "开关",
    "ui": {
        "type": "switch",
        "disabled": false,
        "checked": true,
        "size": "default",
        "checkedChildren": "内容"
    },
    "fieldKey": "switch_J9UiEf"
}
```

:::info 提示

组件具体配置依赖于所使用的UI库，DripForm仅作示例，具体可参考UI库对应文档。

:::

## API
| 参数        | 说明     | 类型     |  是否必填 | 默认值 |
|:-----------|:--------------------------------------------------------------|:--------|:---|:---|
| filedKey      | 表单数据存放字段（必须唯一）| string   | 必填 | - |
| checked | 指定当前是否选中              | boolean | 非必填 | false |
| checkedChildren | 选中时展示的内容 | ReactNode | 非必填 | - |
| disabled      | 值为 `true` 时，开关为禁用状态 | boolean   | 非必填 | false |
| onChange      | 变化时回调函数 | (function(checked: boolean, event: Event)   |  非必填 | - |
| size | 开关大小，可选值：`default` `small`| string | 非必填 | default |
