## 配置
该字段配置在`schema`中：

```json title="schema" {12,13,14}
{
      "type": "array",
      "title": "时间选择框",
      "ui": {
        "type": "timePicker",
        "range": true,
        "use12Hours": true,
        "showNow": true,
        "disabled": true
      },
      "fieldKey": "timePicker_YTyXPD"
    }
```

:::info 提示

组件具体配置依赖于所使用的UI库，DripForm仅作示例，具体可参考UI库对应文档。

:::

## API
| 参数        | 说明     | 类型     |  是否必填 | 默认值 |
|:-----------|:--------------------------------------------------------------|:--------|:---|:---|
| filedKey      | 表单数据存放字段（必须唯一）| string   | 必填 | - |
| allowClear      | 	可以点击清除图标删除内容 | boolean | 必填 | true |
| disabled      | 值为 `true` 时，时间选择器为禁用状态 | boolean   | 非必填 | false |
| format      | 时间格式 | string   | 非必填 | `HH:mm:ss`|
| onChange      | 时间发生变化的回调 | function(time: moment, timeString: string): void   |  非必填 | - |
| placeholder      | 占位文本 | string   | 非必填 | `请选择时间`|
| range      | 是否为范围选择器 | boolean   | 非必填 | false |
| showNow      | 面板是否显示“此刻”按钮 | boolean   | 非必填 | false |
| use12Hours      | 使用 12 小时制，为 `true` 时 `format` 默认为 `HH:mm:ss a`  | boolean   | 非必填 | false |
